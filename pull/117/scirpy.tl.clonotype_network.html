scirpy.tl.clonotype_network¶

scirpy.tl.clonotype_network(adata, *, min_size=1, layout='fr', layout_kwargs=None, neighbors_key='tcr_neighbors', key_clonotype_size='clonotype_size', key_added='X_clonotype_network', inplace=True, random_state=42)¶

Build the clonotype network for plotting

Parameters
min_size : intint (default: 1)

Only show clonotypes with at least min_size cells.

layout : strstr (default: 'fr')

The layout algorithm to use. Can be anything supported by igraph.layout() or “components” to layout all connected components individually. See scirpy._util._graph.layout_componets() for more details.

layout_kwargs : dict, NoneOptional[dict] (default: None)

Will be passed to the layout function

neighbors_key : strstr (default: 'tcr_neighbors')

Key under which the neighborhood graph is stored in adata.uns.

key_clonotype_size : strstr (default: 'clonotype_size')

Key under which the clonotype size information is stored in adata.obs

key_added : strstr (default: 'X_clonotype_network')

Key under which the layout coordinates will be stored in adata.obsm.

inplace : boolbool (default: True)

If true, store the coordinates in adata.obsm, otherwise return them.

random_state

Random seed set before computing the layout.

Return type

None, ndarrayOptional[ndarray]

Returns

  • Depending on the value of inplace returns either nothing or the computed

  • coordinates.
