API¶

Input/Output¶

read_h5ad(filename[, backed, as_sparse, …])

Read .h5ad-formatted hdf5 file.

read_10x_vdj(path[, filtered])

Read TCR data from 10x Genomics cell-ranger output.

read_tracer(path)

Read data from TraCeR.

Datasets: datasets¶

wu2020()

Return the dataset from [Wu2020] as AnnData object.

wu2020_3k()

Return the dataset from [WMdA+20] as AnnData object, downsampled to 3000 TCR-containing cells.

Preprocessing: pp¶

merge_with_tcr(adata, adata_tcr, *[, how, …])

Integrate the TCR AnnData into an existing AnnData object with transcriptomics data.

tcr_dist(unique_seqs, *[, metric, cutoff, …])

calculate the sequence x sequence distance matrix

tcr_neighbors(adata, *[, metric, cutoff, …])

Construct a cell x cell neighborhood graph based on CDR3 sequence similarity.

Tools: tl¶

define_clonotypes(adata, *[, partitions, …])

Define clonotypes based on cdr3 distance.

clonotype_network(adata, *[, min_size, …])

Build the clonotype network for plotting

chain_pairing(adata, *[, inplace, key_added])

Associate categories to cells based on how many TRA and TRB chains they have.

clonal_expansion(adata, *[, target_col, …])

Adds a column to obs which clonotypes are expanded.

summarize_clonal_expansion(adata, groupby, *)

Summarizes clonal expansion by a grouping variable.

alpha_diversity(adata, groupby, *[, …])

Computes the alpha diversity of clonotypes within a group.

group_abundance(adata, groupby[, …])

Creates summary statsitics on how many cells belong to a certain category within a certain group.

spectratype(adata[, groupby, combine_fun, …])

Show the distribution of CDR3 region lengths.

repertoire_overlap(adata, groupby, *[, …])

Compute distance between cell groups based on clonotype overlap.

clonotype_imbalance(adata, replicate_col, …)

Aims to find clonotypes that are the most enriched or depleted in a category.

Plotting: pl¶

alpha_diversity(adata, groupby, *[, …])

Plot the alpha diversity per group.

clonal_expansion(adata, groupby, *[, …])

Plots the the number of cells in target_col that fall into a certain count bin for each group in group_by.

group_abundance(adata, groupby[, …])

Plots how many cells belong to each clonotype.

spectratype(adata[, groupby, combine_fun, …])

Show the distribution of CDR3 region lengths.

vdj_usage(adata, *[, target_cols, fraction, …])

Creates a ribbon plot of the most abundant VDJ combinations in a given subset of cells.

repertoire_overlap(adata, groupby, *[, …])

Visualizes overlap betwen a pair of samples on a scatter plot or all samples on a heatmap or draws a dendrogram of samples only.

clonotype_imbalance(adata, replicate_col, …)

Aims to find clonotypes that are the most enriched or depleted in a category.

clonotype_network(adata, *[, color, …])

Plot the clonotype network

embedding(adata, basis, *[, color, …])

A customized wrapper to the sc.pl.embedding() function.
