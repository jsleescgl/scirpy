scirpy.pp.tcr_neighbors¶

scirpy.pp.tcr_neighbors(adata, *, metric='alignment', cutoff=2, receptor_arms='all', dual_tcr='primary_only', key_added='tcr_neighbors', sequence='aa', inplace=True, n_jobs=None)¶

Construct a cell x cell neighborhood graph based on CDR3 sequence similarity.

Parameters
adata : AnnDataAnnData

annotated data matrix

metric : Literal_[identity, alignment, levenshtein]Literal_[identity, alignment, levenshtein] (default: 'alignment')

“identity” = Calculate 0/1 distance based on sequence identity. Equals a

cutoff of 0.

”alignment” - Calculate distance using pairwise sequence alignment

and BLOSUM62 matrix

”levenshtein” - Levenshtein edit distance

cutoff : intint (default: 2)

Two cells with a distance <= the cutoff will be connected. If cutoff = 0, the CDR3 sequences need to be identical. In this case, no alignment is performed.

receptor_arms : Literal_[TRA, TRB, all, any]Literal_[TRA, TRB, all, any] (default: 'all')

“TRA” - only consider TRA sequences “TRB” - only consider TRB sequences “all” - both TRA and TRB need to match “any” - either TRA or TRB need to match

dual_tcr : Literal_[primary_only, any, all]Literal_[primary_only, any, all] (default: 'primary_only')

“primary_only” - only consider most abundant pair of TRA/TRB chains “any” - consider both pairs of TRA/TRB sequences. Distance must be below cutoff for any of the chains. “all” - consider both pairs of TRA/TRB sequences. Distance must be below cutoff for all of the chains.

key_added : strstr (default: 'tcr_neighbors')

dict key under which the result will be stored in adata.uns["scirpy"] when inplace is True.

sequence : Literal_[aa, nt]Literal_[aa, nt] (default: 'aa')

Use amino acid (aa) or nulceotide (nt) sequences to define clonotype?

inplace : boolbool (default: True)

If True, store the results in adata.uns. If False, returns the results.

n_jobs : int, NoneOptional[int] (default: None)

Number of cores to use for alignment and levenshtein distance.

Return type

Tuple[csr_matrix, csr_matrix], NoneOptional[Tuple[csr_matrix, csr_matrix]]

Returns

  • connectivities – weighted adjacency matrix

  • dist – cell x cell distance matrix with the distances as computed according to metric offsetted by 1 to make use of sparse matrices.
