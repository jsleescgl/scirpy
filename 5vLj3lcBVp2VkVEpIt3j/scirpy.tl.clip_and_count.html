scirpy.tl.clip_and_count¶

scirpy.tl.clip_and_count(adata, groupby, target_col, *, clip_at=3, fraction=True)¶

Counts the number of identical entries in target_col for each group in group_by.

Ignores NaN values.

Parameters
adata : AnnDataAnnData

AnnData object to work on

groupby : strstr

Group by this column from obs

target_col : strstr

Column to count on.

clip_at : intint (default: 3)

All entries in target_col with more copies than clip_at will be summarized into a single group.

fraction : boolbool (default: True)

If True, compute fractions rather than reporting abosolute numbers.

Returns

Return type

Dictionary with counts/fractions per group
